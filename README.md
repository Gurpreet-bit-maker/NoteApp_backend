#Notepad App Backend

This is the Node.js + Express backend for the Notepad App.
It handles user authentication, notes CRUD operations, and JWT-based sessions.

#Features

User signup & login with hashed passwords (bcrypt)
JWT authentication with cookies
Add, edit, delete, and read notes
MongoDB integration (via Mongoose)
CORS enabled for frontend communication

#Tech Stack

Node.js & Express.js
MongoDB & Mongoose
bcrypt for password hashing
jsonwebtoken for authentication
dotenv for environment variables
cookie-parser & cors
